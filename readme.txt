== Description ==
Play the video source which from youku iqiyi and so on without advertising. 

调用ckplayer播放器实现优酷、爱奇艺、音悦台等视频源无广告播放。

插件特色    
    1、调用位于SAE的flash视频播放器，加载响应速度快；    
    2、支持优酷，爱奇艺，56视频等多处视频源，实现无广告播放；    
    3、支持清晰度切换，针对不同带宽自由选择画质。    
        

== Installation ==

可以通过以下两种方法的其中一种来安装ckplayer without ad 插件：

1. 将下载的文件解压缩，然后将`ckplayer_without_ad`文件夹 上传到 `/wp-content/plugins/`目录，在插件后台启用即可

2. 直接在后台-安装插件，搜索'ckplayer without ad'，按照提示安装启用
